Click/Tap on character to open Menu
Escape to Quit

Keyboard Controls:
Move Left - Left Arrow/A
Move Right - Right Arrow/D
Jump - Up Arrow/W
Double Jump - Up Arrow/W while in air (Must have Red active)
Cling - Jump against a wall (Must have Green active)
Dash - Space Bar (Must have Blue active)

Mobile Controls:
Move Left - Hold above Left Arrow UI Element
Move Right - Hold above Right Arrow UI Element
Jump - Hold above Up Arrow UI Element
Double Jump - Jump while in air (Must have Red active)
Cling - Jump against a wall (Must have Green active)
Dash - Swipe (Must have Blue active)
